# NetRelay
NetRelay is a simple network relay tool forwarding the shell command to the destination terminal.

## Usage

**For relay terminal**

```bash
python3 relay.py --src=<sourceAddr>
```

**For client terminal**

```bash
python3 client.py --dst=<destinationAddr>
```
